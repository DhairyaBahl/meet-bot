# Disable Page Visibility API
<a href="https://www.buymeacoffee.com/marvinschopf" target="_blank"><img src="https://www.buymeacoffee.com/assets/img/custom_images/orange_img.png" alt="Buy Me A Coffee" style="height: 41px !important;width: 174px !important;box-shadow: 0px 3px 2px 0px rgba(190, 190, 190, 0.5) !important;-webkit-box-shadow: 0px 3px 2px 0px rgba(190, 190, 190, 0.5) !important;" ></a>

Add-on to disable the Page Visibility API. 

This prevents e.g. video conferencing systems from tracking whether you are currently in another window.
